# test_for_roox
## Тестовое задание для RooX 

```
git clone https://github.com/iurii-anfinogenov/test_for_roox.git
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
# для запуска тестов 
pytest
# Для запуска сервера
uvicorn main:app --reload

```
Сервер запустится на 8000 порту.
